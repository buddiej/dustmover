# dustmover
sensor sytem for detection of position and accelaration with data logging on SD-Card

*Usage of*
- SD-Card (for logging)
- ADXL 345 accelration sensor

# ADXL345 (I2C interface used) (VDD 3.3V)
- Int1 (connected to GPIO27)
- Int2 (connected to GPIO2)
- SCL (GPIO22)
- SDA (GPIO21)
